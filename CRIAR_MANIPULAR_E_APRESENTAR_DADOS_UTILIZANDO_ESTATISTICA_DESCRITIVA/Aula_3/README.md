# 📊 Aula 03 – Estatística Descritiva com Pandas e NumPy

Nesta aula foram apresentados os conceitos de **Estatística Descritiva** utilizando as bibliotecas **Pandas** e **NumPy**. Foram exploradas técnicas de preparação de dados, criação de novas colunas e cálculo de medidas estatísticas para análise de conjuntos de dados.

Também foi introduzido o uso de **Arrays NumPy**, destacando sua eficiência para operações matemáticas e estatísticas em grandes volumes de dados.

## 📚 Conteúdos abordados:

- Importação das bibliotecas:
  - Pandas
  - NumPy
  - OS
- Leitura de arquivos CSV utilizando `read_csv()`.
- Visualização dos dados utilizando `head()`.
- Preparação e transformação de dados.
- Criação de novas colunas em DataFrames.
- Conversão de Series do Pandas para Arrays NumPy utilizando `np.array()`.
- Cálculo de medidas de tendência central:
  - Média (`np.mean()`)
  - Mediana (`np.median()`)
- Cálculo de medidas de posição utilizando quartis:
  - Primeiro Quartil (Q1)
  - Segundo Quartil (Q2)
  - Terceiro Quartil (Q3)
- Seleção de colunas específicas para análise.

## 🧩 Exercícios desenvolvidos:

Durante a aula foram desenvolvidos exemplos práticos como:

- Importação de uma planilha de custos em formato CSV.
- Criação da coluna **Custo Total (R$)** considerando:
  - Preço de compra.
  - Impostos.
  - Frete.
  - Taxa operacional.
- Conversão da coluna de custos para um Array NumPy.
- Cálculo da média dos custos dos produtos.
- Cálculo da mediana dos custos.
- Identificação dos quartis para análise da distribuição dos dados.
- Exibição das informações relevantes para apoio à tomada de decisão.

## 🎯 Objetivo da aula:

Aprender a preparar e transformar dados para análise estatística, utilizando o Pandas para manipulação de DataFrames e o NumPy para realizar cálculos de medidas de tendência central e medidas de posição, desenvolvendo a base necessária para análises descritivas de dados.
