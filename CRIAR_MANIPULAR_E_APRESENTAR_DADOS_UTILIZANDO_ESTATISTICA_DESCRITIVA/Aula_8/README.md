# 📊 Aula 08 – Análise da Distribuição dos Dados e Visualização Estatística

Nesta aula foram aprofundados os conceitos de análise da distribuição dos dados utilizando **Pandas**, **NumPy** e **Matplotlib**. A partir de uma base de dados pública do **Instituto de Segurança Pública do Rio de Janeiro (ISP-RJ)**, foram aplicadas medidas estatísticas para compreender o comportamento da distribuição, identificar outliers e construir diferentes tipos de gráficos para facilitar a interpretação dos resultados.

## 📚 Conteúdos abordados:

- Importação e tratamento de dados utilizando Pandas.
- Agrupamento e ordenação de dados.
- Conversão de Series para Arrays NumPy.
- Revisão das medidas de tendência central:
  - Média.
  - Mediana.
- Revisão das medidas de posição:
  - Quartis (Q1, Q2 e Q3).
- Revisão das medidas de dispersão:
  - Amplitude Total.
  - Intervalo Interquartil (IQR).
- Identificação de outliers superiores e inferiores.
- Medidas de distribuição:
  - Assimetria (`skew()`).
  - Curtose (`kurtosis()`).
- Construção de histogramas.
- Construção de boxplots.
- Criação de gráficos de barras horizontais.
- Organização de múltiplos gráficos utilizando `subplot()`.
- Elaboração de um painel contendo indicadores estatísticos e visualizações gráficas.

## 🧩 Exercícios desenvolvidos:

Durante a aula foram desenvolvidos exemplos práticos como:

- Análise estatística dos roubos de veículos por município.
- Identificação dos municípios com maiores e menores índices de ocorrência.
- Cálculo da média, mediana, quartis e amplitude dos dados.
- Identificação de possíveis outliers utilizando o método IQR.
- Cálculo da assimetria para avaliar o comportamento da distribuição.
- Cálculo da curtose para analisar a concentração dos dados.
- Construção de histogramas para visualizar a frequência das ocorrências.
- Construção de boxplots para identificação visual da dispersão e dos outliers.
- Desenvolvimento de um dashboard estatístico utilizando gráficos e indicadores.

## 🎯 Objetivo da aula:

Compreender como interpretar a distribuição de conjuntos de dados por meio de medidas estatísticas e visualizações gráficas, utilizando histogramas, boxplots, assimetria e curtose para identificar padrões, dispersões e possíveis anomalias nos dados.
