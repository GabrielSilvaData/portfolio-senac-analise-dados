# 📊 Aula 01 – Introdução às Séries do Pandas

Nesta aula foram apresentados os conceitos iniciais da biblioteca **Pandas**, explorando as diferenças entre as estruturas de dados nativas do Python e as estruturas utilizadas para análise de dados. Também foram abordadas boas práticas de organização de projetos Python utilizando os arquivos **.gitignore** e **requirements.txt**.

## 📚 Conteúdos abordados:

- Introdução à biblioteca Pandas.
- Instalação e importação da biblioteca.
- Diferença entre listas do Python e Series do Pandas.
- Criação de listas em Python.
- Criação de objetos `Series`.
- Definição de índices personalizados.
- Associação de índices aos valores armazenados.
- Identificação do tipo de objeto utilizando `type()`.
- Organização de dados utilizando estruturas próprias para análise.
- Utilização do arquivo `.gitignore` para ignorar arquivos e diretórios no Git.
- Utilização do arquivo `requirements.txt` para registrar as dependências do projeto.

## 🧩 Exercícios desenvolvidos:

Durante a aula foram desenvolvidos exemplos práticos como:

- Criação de listas contendo:
  - Produtos.
  - Quantidade em estoque.
- Conversão de listas para Series do Pandas.
- Criação de índices personalizados utilizando o nome dos produtos.
- Visualização dos dados utilizando a estrutura Series.
- Configuração do arquivo `.gitignore` para evitar o versionamento de arquivos desnecessários.
- Criação do arquivo `requirements.txt` contendo as bibliotecas utilizadas no projeto.

## 🎯 Objetivo da aula:

Compreender as diferenças entre as listas do Python e as Series do Pandas, aprender a organizar dados utilizando estruturas do Pandas e aplicar boas práticas na organização de projetos Python com o uso dos arquivos `.gitignore` e `requirements.txt`.
