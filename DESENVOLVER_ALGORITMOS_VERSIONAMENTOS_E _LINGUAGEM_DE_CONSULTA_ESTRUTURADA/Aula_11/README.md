# 🐍 Aula 11 – Introdução ao Pandas e Integração com Banco de Dados

Nesta aula foram apresentados os conceitos iniciais da biblioteca **Pandas**, utilizada para manipulação e análise de dados em Python. Foram exploradas as principais formas de criar e manipular **DataFrames**, realizar leitura de arquivos CSV e conectar aplicações Python a um banco de dados MySQL.

Também foram apresentados métodos para obter estatísticas de conjuntos de dados e a utilização da biblioteca **SQLAlchemy** para integração entre Python e banco de dados.

## 📚 Conteúdos abordados:

- Instalação e importação da biblioteca **Pandas**.
- Criação de dicionários em Python.
- Conversão de dicionários em **DataFrames**.
- Exibição de dados completos e de colunas específicas.
- Leitura de arquivos CSV utilizando `read_csv()`.
- Configuração do separador de arquivos CSV com o parâmetro `sep`.
- Métodos estatísticos do Pandas:
  - `max()`
  - `min()`
  - `mean()`
  - `sum()`
- Visualização das primeiras linhas utilizando `head()`.
- Instalação das bibliotecas:
  - `sqlalchemy`
  - `pymysql`
- Conexão entre Python e banco de dados MySQL utilizando `create_engine()`.
- Leitura de tabelas do banco de dados utilizando `read_sql()`.

## 🧩 Exercícios desenvolvidos:

Durante a aula foram desenvolvidos exemplos práticos como:

- Criação de um DataFrame contendo:
  - Cargos.
  - Salários.
- Consulta de colunas específicas do DataFrame.
- Importação de um arquivo CSV contendo produtos.
- Cálculo de estatísticas como:
  - Maior preço.
  - Menor preço.
  - Média dos preços.
  - Soma total dos preços.
- Conexão com um banco de dados MySQL e leitura de uma tabela utilizando Pandas.
- Criação de um DataFrame com informações de filmes contendo:
  - Título.
  - Ano de lançamento.
  - Gênero.

## 🎯 Objetivo da aula:

Aprender os fundamentos da biblioteca **Pandas** para manipulação e análise de dados, realizar importação de arquivos CSV, obter informações estatísticas de conjuntos de dados e integrar aplicações Python com bancos de dados relacionais utilizando MySQL.
