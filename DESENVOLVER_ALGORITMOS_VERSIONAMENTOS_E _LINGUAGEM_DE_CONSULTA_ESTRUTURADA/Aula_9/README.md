# 🐍 Aula 09 – Funções, Modularização e Bibliotecas em Python

# Nesta aula foram aprofundados os conceitos de funções, explorando como organizar programas maiores utilizando menus de opções, separação de responsabilidades e reutilização de código através da modularização.

# Também foram apresentadas bibliotecas nativas do Python, como random e os, demonstrando como utilizar recursos externos para complementar as funcionalidades dos programas.

# 📚 Conteúdos abordados:

# Criação de programas utilizando funções específicas para cada tarefa.

# Funções com parâmetros e retorno de valores (return).

# Organização do código através da divisão de responsabilidades.

# Criação de menus interativos utilizando:

# Entrada de dados com input()

# Estrutura de decisão match/case.

# Reutilização de funções em diferentes partes do programa.

# Conceito de modularização:

# Separação de funções em arquivos externos

# Importação de funções utilizando import.

# Uso de bibliotecas Python:

# random para geração de números aleatórios

# os para interação com comandos do sistema.

# Validação de tipos utilizando isinstance().

# Tratamento de diferentes tipos de retorno (números e textos).

# 🧩 Exercícios desenvolvidos:

# Durante a aula foram desenvolvidos exemplos práticos como:

# Menu de operações matemáticas:

# Dobro

# Triplo

# Quadrado

# Metade.

# Calculadora utilizando funções:

# Soma

# Subtração

# Multiplicação

# Divisão.

# Organização de projetos utilizando arquivos auxiliares para armazenar funções e facilitar a manutenção do código.

# 🎯 Objetivo da aula:

# Aprender a estruturar programas de forma mais profissional utilizando funções e módulos, reduzindo repetição de código e criando aplicações mais organizadas, reutilizáveis e fáceis de manter.
