🐍 Aula 01 – Introdução ao Python
📚 Resumo
Nesta primeira aula foram apresentados os conceitos básicos da linguagem Python, fundamentais para o desenvolvimento de programas e para a análise de dados.

Conteúdos abordados
Utilização da função print() para exibir informações na tela.
Criação de comentários (#) para documentar o código.
Atalhos úteis do VS Code para aumentar a produtividade.
Criação e utilização de variáveis.
Conceito de atribuição de valores utilizando o operador =.
Uso de f-strings para formatar textos com variáveis.
Operadores matemáticos:
Soma (+)
Subtração (-)
Multiplicação (*)
Divisão (/)
Módulo (%)
Divisão inteira (//)
Atualização de valores em variáveis.
Reutilização de variáveis em novos cálculos (acumulação).
🎯 Objetivo da aula
Compreender a estrutura básica da linguagem Python, aprendendo a criar variáveis, manipular valores, realizar operações matemáticas e exibir informações na tela, formando a base para os próximos estudos em programação e análise de dados.
