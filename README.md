# 💻 Unidade Curricular – Desenvolver Algoritmos, Versionamento e Linguagem de Consulta Estruturada

Nesta unidade curricular foram desenvolvidas as bases da programação e do desenvolvimento de soluções para análise de dados. Durante as aulas foram estudados conceitos de lógica de programação, algoritmos, versionamento de código utilizando Git e GitHub e consultas em bancos de dados relacionais utilizando SQL.

Além disso, foram utilizadas bibliotecas do Python para manipulação de dados e integração com bancos de dados, preparando a base necessária para o desenvolvimento de projetos voltados à área de Dados.

- Estruturas de repetição (`for` e `while`).
- Listas e dicionários.
- Funções.
- Modularização.
- Bibliotecas Python.
- Tratamento de exceções (`try`, `except`, `else` e `finally`).
- Manipulação de arquivos CSV.
- Introdução ao Pandas.

### 🌿 Git e GitHub
- Versionamento de código.
- Criação de repositórios.
- Commits.
- Branches.
- Push e Pull.
- Fluxo de desenvolvimento utilizando Git.

### 🗄️ SQL e Banco de Dados
- Conceitos de Banco de Dados Relacional.
- Linguagem SQL.
- Consultas em tabelas.
- Integração entre Python e MySQL.
- Leitura de tabelas utilizando Pandas.
- Conexão com banco de dados utilizando SQLAlchemy.

## 🧩 Projetos e exercícios desenvolvidos

- Desenvolvimento de algoritmos em Python.
- Exercícios de lógica de programação.
- Manipulação de arquivos CSV.
- Integração entre Python e banco de dados MySQL.

# 📊 Unidade Curricular – Criar, Manipular e Apresentar Dados Utilizando Estatística Descritiva

Esta unidade curricular teve como foco a aplicação de técnicas de Estatística Descritiva utilizando Python para transformar dados em informações relevantes. Durante as aulas foram utilizadas bibliotecas como **Pandas**, **NumPy**, **Matplotlib** e **Polars** para importar, tratar, analisar e visualizar bases de dados reais.

Foram desenvolvidas análises utilizando bases públicas, como os dados do **Instituto de Segurança Pública do Rio de Janeiro (ISP-RJ)** e do **Portal da Transparência (Novo Bolsa Família)**, permitindo aplicar conceitos estatísticos em cenários práticos de análise de dados.

## 📚 Conteúdos abordados

### 📂 Manipulação de Dados
- Importação de arquivos CSV.
- Tratamento e limpeza de dados.
- Seleção e filtragem de colunas e registros.
- Agrupamento de dados (`groupby`).
- Ordenação de informações.
- Concatenação de DataFrames.
- Conversão entre tipos de dados.
- Leitura e escrita de arquivos Parquet.

### 📈 Estatística Descritiva
- Média.
- Mediana.
- Quartis (Q1, Q2 e Q3).
- Amplitude Total.
- Intervalo Interquartil (IQR).
- Identificação de outliers.
- Variância.
- Desvio Padrão.
- Coeficiente de Variação.
- Assimetria.
- Curtose.

### 📊 Visualização de Dados
- Gráficos de barras.
- Gráficos de barras horizontais.
- Histogramas.
- Boxplots.
- Painéis estatísticos utilizando `subplot()`.
- Exibição de indicadores estatísticos em gráficos.

### ⚡ Processamento de Grandes Volumes de Dados
- Introdução ao Polars.
- Comparação de desempenho entre Pandas e Polars.
- Processamento em paralelo (Multithreading).
- Lazy Loading (`scan_parquet`).
- Planos de execução.
- Otimização de consultas.
- Conversão de arquivos CSV para Parquet.

## 🧩 Projetos e exercícios desenvolvidos

- Análise estatística de ocorrências criminais do Estado do Rio de Janeiro.
- Identificação dos municípios com maiores e menores índices de criminalidade.
- Cálculo de medidas de tendência central e dispersão.
- Identificação de outliers utilizando o método IQR.
- Construção de dashboards estatísticos utilizando Matplotlib.
- Processamento de múltiplos arquivos do Bolsa Família.
- Comparação de desempenho entre Pandas e Polars.
- Geração e leitura de arquivos Parquet.
- Processamento otimizado utilizando Lazy Evaluation.

## 🛠️ Tecnologias utilizadas

- Python
- Pandas
- NumPy
- Matplotlib
- Polars
- Parquet

## 🎯 Objetivo da unidade curricular

Desenvolver competências para coletar, manipular, organizar, analisar e apresentar dados por meio de técnicas de Estatística Descritiva, utilizando ferramentas da linguagem Python para transformar dados em informações relevantes, produzir visualizações estatísticas e otimizar o processamento de grandes volumes de dados.
