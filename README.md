🐍 Aula 01 – Introdução ao Python
Nesta aula foram apresentados os conceitos fundamentais da linguagem Python.

Conteúdos abordados:

Uso da função print();
Comentários no código (#);
Variáveis e atribuição de valores;
Tipos de dados básicos;
Formatação de texto com f-strings;
Operadores matemáticos (+, -, *, /, % e //);
Atualização e reutilização de variáveis em cálculos.
Objetivo: compreender a estrutura básica da linguagem Python e desenvolver a lógica necessária para os próximos conteúdos do curso.
